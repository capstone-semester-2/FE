# ReVoice 프로젝트 요약

## 프로젝트 개요

어눌한 발음을 하는 언어장애인을 위한 AI 기반 음성 번역 서비스.
사용자의 어눌한 음성을 입력받아 자연스럽고 명확한 문장으로 변환해주는 기능을 제공하며, 수화 연동 및 기록/즐겨찾기 기능도 함께 제공.

---

## 핵심 기능

### 1. 어눌한 말 번역 서비스

* 사용자의 음성을 녹음하고 AI 모델로 전달 → 명확한 문장 반환
* 프론트: 마이크 녹음, 변환 결과 표시, TTS 재생
* 백엔드: Presigned URL + S3 업로드, AI서버로 전달
* AI: Kospeech 기반 모델, 구음장애 음성 학습

### 2. 수화 영상 연동 기능

* 번역된 문장의 단어를 클릭하면 해당 단어의 수화 영상을 재생
* 수화 데이터는 YouTube 링크로 제공 예정

### 3. 수화 백과사전

* 사용자가 단어를 검색해 수화 영상을 조회
* 검색창 + 영상 리스트 UI

### 4. 기록 기능

* 음성 및 번역 결과 저장/조회
* 리스트 형태로 무한 스크롤 구현

### 5. 즐겨찾기 및 단어 사용 통계

* 자주 사용하는 단어를 분석하여 통계 제공
* 북마크한 수화 영상 목록 관리

---

## 프론트엔드 개발 환경

* 기술 스택: React (JS 또는 TS)
* 라우팅: 탭 구조 기반 화면 (음성변환 / 기록 / 즐겨찾기 / 백과사전)
* 상태관리: Context API 또는 Redux 또는 Zustand
* 음성 녹음: MediaRecorder API 또는 외부 라이브러리 사용 가능
* 초기 개발: Mock 데이터 사용, 이후 실제 API 연동

---

## 백엔드 연동 구조

* 음성 업로드: Presigned URL 방식 → S3에 직접 업로드
* 번역 요청 결과: SSE(Server-Sent Events)로 실시간 푸시
* 수화 영상: YouTube 링크 기반 streaming 제공

---

## 개발 흐름 요약 (프론트)

1. 화면별 컴포넌트 구현 (Figma 기준)
2. 녹음 기능 구현 (MediaRecorder 기반 커스텀 훅 등)
3. Mock API로 기능 테스트
4. Context/Redux로 전역 상태 설계
5. API 명세 완료 후 실제 백엔드 연동
6. 최종 배포: S3 + CloudFront 기반 정적 사이트 호스팅
